1. Grid size （網格大小）
作用： 控制 3D 重建時的網格精度
 
Grid size	精度	    速度 點數量	     用途
10-20	🟢 超高精度	🔴 很慢	~15,000+	精細掃描、高品質模型
30-40	🟡 標準精度	🟡 正常	~3,840	    推薦使用
50-70	🟠 低精度	🟢 快速	~1,500	    快速預覽、低端硬體
80-100	🔴 極低精度	🟢 很快	~500	    速度優先測試
